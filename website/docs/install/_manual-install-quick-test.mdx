### Quick test {#manual-install-quick-test}

To check if the xPack pkg-config installed manually starts properly, use something like:

<Tabs groupId="operating-systems">
<TabItem value="windows" label="Windows" default>

<CodeBlock language="console"> {
`C:\\> %USERPROFILE%\\AppData\\Roaming\\xPacks\\pkg-config\\xpack-pkg-config-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }\\bin\\TODO.exe --version
TODO ${ customField('upstreamVersion') }`
} </CodeBlock>

</TabItem>
<TabItem value="macos" label="macOS">

<CodeBlock language="console"> {
`$ ~/Library/xPacks/pkg-config/${ customField('upstreamVersion') }-${ customField('xpackSubversion') }/bin/TODO --version
TODO ${ customField('upstreamVersion') }`
} </CodeBlock>

</TabItem>
<TabItem value="linux" label="GNU/Linux">

<CodeBlock language="console"> {
`$ ~/.local/xPacks/pkg-config/xpack-pkg-config-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }/bin/TODO --version
TODO ${ customField('upstreamVersion') }`
} </CodeBlock>

</TabItem>
</Tabs>
